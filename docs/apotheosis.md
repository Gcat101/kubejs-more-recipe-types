### Apotheosis

#### (Infusion) Enchanting

```js
global.mrt.apotheosis.enchanting(event, <Output item>, <Input item>, <Level required DISPLAY (3 by default)>, [[<Eterna min>, <Eterna max> (both 20 by default)], [<Quanta min (20 by default)>, <Quanta max (min + 20 by default))>], [<Arcana min (20 by default)>, <Arcana max (min + 20 by default))>]], <id>)
```

#### fletching

```js
global.mrt.apotheosis.fletching(event, <Output item>, '{Potion:"minecraft:night_vision"}'), [<Input items>], <id>)
```

#### Spawner Modifier 

```js
global.mrt.apotheosis.spawner_modifier(event, [[<Stat to change id>, <Value to Change it to / by (depending on id)>, <Min value (needed for some id's)>, <Max value (needed for some id's)>], ...], [<Mainhand input item>, <Offhand input item>, <Consume offhand ? (false by default)>], <id>)
```
